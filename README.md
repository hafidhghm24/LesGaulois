# LesGaulois
Apprendre le Java et l'UML avec Astérix & Obélix ⚡️
Pour maîtriser le Java et la conception UML, j'ai décidé d'appliquer la théorie sur un cas concret et ludique, simuler le monde d'Astérix et Obélix.

- Modélisation d'un diagramme de classes complet avec draw.io.
- Polymorphisme pour gérer la logique des batailles.
- Interfaces pour la structuration des scénarios de combat (Bataille, Embuscade).
- Structures de données pour la gestion des équipements (Casque, Bouclier) via des tableaux et des énumérations (enum).

C'est un excellent exercice pour découvrir la POO en Java. Pour ceux que ça intéresse
